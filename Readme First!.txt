Dependencies :
1. Python 3
2. Pip(Python 3' Package Manager)
3. Python 3 Modules :
	- flask
	- datetime
	- pandas
	- numpy
	- sqlite3
	- pprint

Modules Installation :
Command : "pip install flask datetime pandas numpy sqlite3 pprint"

- "cd GoLibrary"
- "python database_generator.py"

- "cd GoLibrary"
